## Descripción

En esta sección se presentan imágenes generadas a partir del análisis de
datos socioeconómicos de países de Sudamérica, obtenidos mediante la API
del Banco Mundial.

Las visualizaciones permiten representar de forma clara indicadores
económicos y sociales, facilitando la comparación entre países y la
identificación de patrones relevantes dentro de la región.

Estas imágenes se utilizan como apoyo al análisis estadístico y a la
presentación de resultados en trabajos académicos y exposiciones
grupales.
